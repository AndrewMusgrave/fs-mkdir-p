# mkdirp

Creates nested directories like `mkdir -p`

## Install

```bash
npm install fs-mkdir-p
```

## Usage

```js
// directory path
const mkdirp = require("fs-mkdir-p");
mkdirp("src\\components\\Modal\\tests");
```

## API

### mkdirp(dir)

| Param | Type   | Default | Description        |
| ----- | ------ | ------- | ------------------ |
| dir   | string |         | direcory to create |
